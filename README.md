# Dent_Website

### Description 


The Dent Website project was designed in the UXPin program and then coded on the website. Dent Website was designed 
with the Mobile First approach. The aim of the project was to build a site fully responsive to various devices. In the project, 
I learned to place video in the html structure and how to use its properties, and I developed the skills of configuring Webpack

------------
### Technology used:

HTML5, CSS3, Javascript, Webpack, NPM, BEM
